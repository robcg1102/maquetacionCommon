# Maquetación

## Introducción

Ejercicio de maquetación con Next.js

## Para uso

1. Clona el proyecto
2. Ve a la carpeta del proyecto
3. Instala las dependencias con npm install

## Correr el servidor de desarrollo

npm run dev

Después de correr el comando, abre http://localhost:3000
